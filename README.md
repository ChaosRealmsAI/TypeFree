# TypeFree

> **归档说明**：本项目将在豆包官方上线语音输入法功能后归档。届时请使用官方输入法获得更好的体验。

**长按 Fn 键说话，松开自动粘贴** — 一个借助豆包 ASR 服务实现的语音输入工具。

## 功能特性

- **Fn 键触发**：按住 Fn 开始录音，松开结束
- **实时识别**：流式 ASR，边说边显示识别结果
- **自动粘贴**：识别完成后自动粘贴到光标位置
- **浮层显示**：实时显示识别文字，跟随焦点窗口
- **后台常驻**：系统托盘运行，支持开机自启

## 系统要求

- macOS 12.0+ (Apple Silicon / Intel)
- 豆包桌面端（会自动以调试模式启动）

## 安装

### 方式一：直接下载

前往 [Releases](../../releases) 下载最新版 `.dmg` 文件，打开后将 TypeFree 拖入「应用程序」文件夹。

首次打开时，右键点击应用 → 选择「打开」（绕过 Gatekeeper）。

### 方式二：从源码构建

```bash
git clone <repo-url>
cd typefree

pnpm install
pnpm tauri dev      # 开发模式
pnpm tauri build    # 构建发布包
```

## 使用指南

1. 启动应用后会自动启动豆包桌面端（调试模式）
2. 首次使用需授权系统权限（输入监控、辅助功能、麦克风）
3. 将光标定位到任意输入框
4. **按住 Fn 键**，开始说话
5. **松开 Fn 键**，文字自动粘贴到光标位置

## 权限设置

如果功能异常，请检查：

```
系统设置 → 隐私与安全性 → 输入监控 → 勾选 TypeFree
系统设置 → 隐私与安全性 → 辅助功能 → 勾选 TypeFree
系统设置 → 隐私与安全性 → 麦克风 → 勾选 TypeFree
```

## 技术栈

- Tauri 2.0 + Rust
- 豆包 ASR（通过 Chrome DevTools Protocol）
- cpal（音频采集）+ rubato（重采样）
- IOKit（macOS 键盘监听）
